## 六、GitHub优质C++面试资源

在准备C++面试时，GitHub上有许多优质的开源资源可以参考。以下是一些比较全面的C++面试题开源库，涵盖了基础知识、算法、设计模式、高级特性等多个方面：

### 1. Cpp-Questions
- **链接**：[https://github.com/giantbranch/Cpp-Questions](https://github.com/giantbranch/Cpp-Questions)
- **主要内容**：包含C++基础、STL、多线程、设计模式、内存管理等方面的面试题和答案
- **特点**：题目分类清晰，答案详细，包含大量示例代码

### 2. CPlusPlusThings
- **链接**：[https://github.com/Light-City/CPlusPlusThings](https://github.com/Light-City/CPlusPlusThings)
- **主要内容**：不仅包含面试题，还涵盖C++核心知识点、实战项目和学习路径
- **特点**：内容系统全面，从入门到高级，适合系统性学习

### 3. Cpp-Interview
- **链接**：[https://github.com/huihut/interview](https://github.com/huihut/interview)
- **主要内容**：包含C++、操作系统、网络、数据结构与算法等多方面面试题
- **特点**：覆盖面广，结合实际开发场景，适合全栈准备

### 4. advanced-cpp-knowledge
- **链接**：[https://github.com/youngyangyang04/advanced-cpp-knowledge](https://github.com/youngyangyang04/advanced-cpp-knowledge)
- **主要内容**：聚焦C++高级特性，包括模板元编程、现代C++特性等
- **特点**：深入讲解C++底层原理，适合进阶学习

### 5. CppGuide
- **链接**：[https://github.com/chenshuo/CppGuide](https://github.com/chenshuo/CppGuide)
- **主要内容**：由陈硕老师维护，包含C++编程规范、最佳实践和面试要点
- **特点**：权威性高，注重实践经验，适合提升工程能力

### 6. Cpp-Primer-Plus-Notes
- **链接**：[https://github.com/jzplp/Cpp-Primer-Plus-Notes](https://github.com/jzplp/Cpp-Primer-Plus-Notes)
- **主要内容**：基于《C++ Primer Plus》的学习笔记和配套面试题
- **特点**：理论与实践结合，适合巩固基础知识

### 7. algorithm-pattern
- **链接**：[https://github.com/greyireland/algorithm-pattern](https://github.com/greyireland/algorithm-pattern)
- **主要内容**：专注于算法与数据结构的面试题，包含C++实现
- **特点**：算法分类清晰，解题思路详细，适合提升算法能力

### 8. modern-cpp-tutorial
- **链接**：[https://github.com/changkun/modern-cpp-tutorial](https://github.com/changkun/modern-cpp-tutorial)
- **主要内容**：现代C++特性详解，包含C++11/14/17/20的新特性
- **特点**：紧跟C++标准发展，适合学习现代C++编程

这些开源库各有侧重，您可以根据自己的需求选择合适的资源进行学习。建议结合实际项目经验，不仅要记住答案，更要理解背后的原理和应用场景。
